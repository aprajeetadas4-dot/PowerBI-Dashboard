# PowerBI-Dashboard
Power BI Dashboard Project

Objective

To create a data visualization dashboard using Power BI and generate business insights through storytelling.

Tools Used

- Power BI
- Superstore Dataset

Visualizations

- Sales by Category
- Sales by Segment
- Sales by State
- Profit by Region
- Profit by Sub-Category
- KPI Cards

Key Insights

- Technology category generated the highest sales.
- West region generated maximum profit.
- Consumer segment contributed the highest sales.
